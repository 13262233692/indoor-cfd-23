## 1. 产品概述

室内空气流动CFD（计算流体力学）模拟Web应用，用户可通过画布自定义绘制房间形状、送风口和回风口位置，设置风速参数，系统自动计算并可视化展示室内速度场和浓度场分布。

- **主要目的**：提供直观易用的室内通风模拟工具，帮助建筑设计师、暖通工程师快速评估室内空气流动情况
- **解决的问题**：传统CFD软件学习成本高、操作复杂，本应用降低使用门槛，实现快速建模与可视化
- **目标用户**：建筑设计师、暖通工程师、科研人员、学生

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 绘制几何模型、设置参数、提交计算、查看结果 |

### 2.2 功能模块
1. **绘图界面**：Canvas画布绘制房间轮廓、送风口、回风口
2. **参数设置**：风速设置、边界条件配置
3. **任务队列**：计算任务提交、状态查询、历史记录
4. **结果可视化**：速度场箭头图、浓度场热力图

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页面 | 工具栏 | 绘图工具选择（房间/送风口/回风口/擦除）、撤销/重做、清空画布 |
| 主页面 | Canvas绘图区 | 鼠标绘制多边形房间、拖拽放置风口、网格显示 |
| 主页面 | 参数面板 | 送风速度、回风速度、流体属性、迭代步数设置 |
| 主页面 | 任务面板 | 提交计算、任务状态显示、进度条、结果切换 |
| 主页面 | 结果展示区 | 速度场箭头图、浓度场热力图、颜色图例、交互查询 |

## 3. 核心流程

用户在Canvas画布上绘制房间多边形轮廓 → 在房间边界上放置送风口和回风口 → 设置送风速度等参数 → 点击"开始计算"提交任务 → 系统将几何数据和参数发送至后端 → 后端加入任务队列并调用Python求解器 → 求解器使用格子Boltzmann方法计算速度场和浓度场 → 计算完成后返回结果数据 → 前端解析结果并在Canvas上绘制箭头图和热力图 → 用户可切换查看不同物理量、调整显示参数

```mermaid

    A["绘制房间形状"] --> B["放置送/回风口"]
    B --> C["设置计算参数"]
    C --> D["提交计算任务"]
    D --> E["后端任务队列"]
    E --> F["LBM求解器计算"]
    F --> G["生成速度场/浓度场"]
    G --> H["箭头图/热力图展示"]
    H --> I["交互查询与分析"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝色（#1e3a5f）科技感，搭配青色（#00d4ff）作为强调色
- **辅助色**：橙色（#ff6b35）表示送风口，蓝色（#4ecdc4）表示回风口
- **背景**：深色主题（#0f1626），符合工程仿真软件的专业感
- **按钮风格**：圆角矩形，轻微阴影，悬停时有亮度变化
- **字体**：JetBrains Mono 作为代码和数值显示，Inter 作为界面文本
- **布局**：三栏布局，左侧工具栏 + 中间画布区 + 右侧参数面板

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 主页面 | 顶部导航 | Logo、应用标题、帮助按钮 |
| 主页面 | 左侧工具栏 | 绘图工具图标（多边形、风口、擦除）、操作按钮（撤销/清空）、网格开关 |
| 主页面 | 中央画布 | 深色网格背景、已绘制的房间形状（白色边框）、送风口（橙色箭头）、回风口（蓝色箭头） |
| 主页面 | 右侧参数面板 | 分组折叠面板：几何信息、边界条件、求解参数、显示设置 |
| 主页面 | 底部状态栏 | 任务状态、计算进度条、当前鼠标坐标、网格尺寸 |

### 4.3 响应性
- 桌面端：三栏固定布局，画布自适应剩余空间
- 平板端：右侧面板可折叠收起，画布占主要空间
- 移动端：垂直布局，工具栏移至顶部，参数面板使用底部抽屉

### 4.4 可视化风格
- **箭头图**：速度矢量用彩色箭头表示，箭头长度与速度成正比，颜色表示速度大小
- **热力图**：浓度分布使用彩虹色阶（蓝-青-绿-黄-红），支持透明度调节
- **图例**：垂直颜色条，显示数值范围，支持鼠标悬停显示精确值
- **动画**：计算完成时结果区域淡入，箭头有轻微呼吸动画表示流动
